# Page 23 — Поиск

## Назначение

Модуль поиска находит доступные активному аккаунту данные приложения: людей, бизнесы, места, предложения, посты и видео. Поиск работает от `activeAccountId` и учитывает приватность/follow graph.

## Входы

- Главная → кнопка поиска в шапке.
- Карта → кнопка поиска в шапке карты.

## UI

`SearchScreen`:

- заголовок `Поиск`;
- строка `Найти людей, места, бизнесы...`;
- фильтры `Все / Люди / Бизнес / Места / Предложения / Посты / Видео`;
- блок `Недавние поиски` при пустой строке;
- блок `Результаты` при запросе от 2 символов;
- пустое состояние `Ничего не найдено`.

## API

- `GET /api/search?q=&type=all|people|business|places|offers|posts|videos`
- `GET /api/search/recent`
- `POST /api/search/recent`
- `DELETE /api/search/recent/:id`
- `DELETE /api/search/recent`

## Переходы

- Люди/бизнес → `PublicProfileScreen`.
- Предложение → `OfferDetailScreen`.
- Место → `MapScreen`.
- Видео → вкладка `Видео`.
- Пост → `PostDetailScreen`.

## Ограничения

- Нет рекламы в поиске.
- Нет AI-поиска.
- Нет демо-результатов.
- Нет импорта контактов.
- Нет платного продвижения.
- Закрытый/followers-only контент не показывается неподписчику.
