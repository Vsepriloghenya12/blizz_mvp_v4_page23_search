# Страница / модуль 2 — Навигация и MainTabs

## Статус
Согласовано: нижнее меню изменено относительно v4 по решению пользователя.

## Нижнее меню

```text
Главная / Видео / + / Карта / Профиль
```

Сообщения не являются пунктом нижнего меню. Основной вход в сообщения — кнопка справа в верхней шапке Главной.

## Переходы

```text
App start
  → GET /api/auth/me
    → session valid: MainTabs
    → no session: Регистрация / Вход

MainTabs
  → Главная: FeedScreen
      → верх справа: MessagesScreen
  → Видео: VideoFeedScreen
  → +: CreateHubScreen / Что создать?
  → Карта: MapScreen
  → Профиль: ProfileScreen(activeAccountId)
```

## Кнопки

| Кнопка | Функция | Переход |
|---|---|---|
| Главная | Открыть ленту | FeedScreen |
| Видео | Открыть вертикальную видеоленту | VideoFeedScreen |
| + | Открыть выбор типа создания | CreateHubScreen |
| Карта | Открыть discovery-карту | MapScreen |
| Профиль | Открыть профиль activeAccountId | ProfileScreen |
| Сообщения | Открыть чаты | MessagesScreen |

## Правила

- `+` не создаёт контент напрямую, а открывает экран “Что создать?”.
- На карте нет кнопки создания контента.
- Профиль строится от `activeAccountId`, а не от `user.isBusiness`.
- Сообщения остаются полноценным модулем, но открываются через верхнюю кнопку, share, Близзы и бизнес-контексты.
- Визуал пока не финализируется; это логическая оболочка приложения.
