# Page 19 — Игры в чатах / Напёрстки

## Назначение

`Напёрстки` — лёгкая чат-игра на внимание внутри личного или группового чата.

Путь:

```text
Сообщения → Личный/групповой чат → + → Игры → Напёрстки
```

## Доступность

Доступно:

- личный чат;
- групповой чат.

Недоступно:

- бизнес-чат;
- нижнее меню;
- карта;
- профиль;
- витрина;
- предложения.

## Логика

1. Пользователь запускает игру через `GamePickerSheet`.
2. Backend создаёт `GameSession type=shells`.
3. В чат добавляется `game_invite`.
4. Игрок проходит 5 раундов.
5. В каждом раунде выбирает один из трёх стаканчиков.
6. Backend случайно определяет позицию шарика.
7. После 5 раундов пользователь отправляет результат в чат.
8. В чат добавляется `game_result`.

## Backend/API

Используется общий игровой слой:

```text
GET /api/games/catalog
POST /api/games/sessions
GET /api/games/sessions/:sessionId
POST /api/games/sessions/:sessionId/answer
POST /api/games/sessions/:sessionId/finish
```
