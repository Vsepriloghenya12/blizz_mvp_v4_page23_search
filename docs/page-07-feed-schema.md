# Страница 7 — Главная / лента

## Назначение

Главная показывает опубликованные посты, строку Близзов, кнопку сообщений и действия под постами. Обновление ленты происходит через свайп вниз / pull-to-refresh, без отдельной кнопки обновления.

## Переход

```text
Нижнее меню → Главная → FeedScreen
```

## Структура

```text
┌────────────────────────────────────┐
│ Близз                         ✉    │
│                                    │
│ Близзы                            │
│ [Ваш Близз +] [логический блок]    │
│                                    │
│ Лента                              
│                                    │
│ Автор                         ⋯    │
│ @username · Геометка               │
│ [Фото]                             │
│ ♡ 0   Коммент. 0   Поделиться  Сохранить
│ Текст поста                        │
│ время                              │
└────────────────────────────────────┘
```

## Кнопки

| Кнопка | Функция | Статус |
|---|---|---|
| Сообщения | Открывает MessagesScreen | работает |
| Свайп вниз | Обновляет ленту через pull-to-refresh | работает |
| Лайк | `POST /api/posts/:postId/like` | работает, optimistic update |
| Сохранить | `POST /api/posts/:postId/save` | работает, optimistic update |
| Комментарии | Будущий CommentsSheet | не реализовано в этом модуле |
| Поделиться | Будущий ShareSheet | не реализовано в этом модуле |
| Ваш Близз + | Будущая stories-механика | не создаёт Близз сейчас |

## Backend

```text
GET /api/posts/feed
POST /api/posts/:postId/like
POST /api/posts/:postId/save
```

## Запреты

- не добавлять отдельную кнопку `Обновить`; обновление только через свайп вниз / pull-to-refresh;
- не добавлять создание поста на Главную;
- не возвращать Близз в `+`;
- не делать полноценные комментарии до отдельного модуля;
- не делать полноценный Share до отдельного модуля;
- не добавлять промо, рекомендации и витрину без данных.
