# Страница 13 — Карта и геометки

## Смысл

Карта — discovery-экран Близз: «Что интересного рядом?». Экран собирает уже созданный контент с геометками и показывает его в одном месте.

## Вход

Нижнее меню → Карта.

## Что показывает

- опубликованные посты с геометкой;
- опубликованные видео с геометкой;
- активные Близзы с геометкой;
- бизнесы с адресом;
- предложения с адресом/геометкой;
- сохранённые объекты пользователя.

## Фильтры

- Все;
- Посты;
- Видео;
- Близзы;
- Бизнес;
- Предложения;
- Сохранённое.

## Кнопки и переходы

| Элемент | Действие |
|---|---|
| Фильтр | Меняет тип объектов карты |
| Метка / объект списка | Открывает нижнюю карточку объекта |
| Предложение → Открыть | Открывает OfferDetailScreen |
| Видео → К видео | Переводит во вкладку Видео |
| Маршрут | Открывает Яндекс.Карты через внешнюю ссылку |
| Сохранить | Работает для постов, видео и предложений через существующие save endpoints |
| Закрыть карточку | Закрывает нижнюю карточку |

## Важные ограничения

- На карте нет кнопки создания контента.
- Карта не показывает live tracking пользователя.
- Контент создаётся только через согласованные сценарии: + / Что создать? или строка Близзов.
- Не-public контент не показывается чужим аккаунтам.
- Финальный Android-вариант должен использовать нативный Yandex MapKit. В этой версии web/dev-экран — технический рабочий слой для проверки данных, фильтров и переходов.

## Backend

- GET /api/map/objects?filter=all|post|video|story|business|offer|saved

Backend сам определяет пользователя и activeAccountId по session token.
