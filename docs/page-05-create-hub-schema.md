# Близз MVP v4 — страница 5: + / Что создать?

## Статус

Логика согласована и реализована как продолжение версии 4.

## Назначение

Экран открывается по центральному пункту нижнего меню `+` и выбирает тип создания от имени `activeAccountId`.

Экран не является редактором, лентой, витриной, профилем или настройками.

## Главное изменение относительно исходного v4

Создание `Близза` убрано из экрана `+ / Что создать?`.

Близз будет создаваться позже отдельной stories-механикой из строки Близзов пользователя, по логике Instagram Stories:

```text
Главная → строка Близзов → мой Близз / + → Создать Близз
```

## Доступные действия

### Личный аккаунт

```text
Пост
Видео
```

### Бизнес owner / admin / smm

```text
Пост
Видео
Предложение
```

### Бизнес messages

```text
У вас доступ только к сообщениям
[Открыть сообщения]
```

## Схема

```text
┌────────────────────────────────────┐
│ Близз                              │
│ Что создать?                       │
│ Пользователь / бизнес ▼            │
│                                    │
│ Пост          →                    │
│ Видео         →                    │
│ Предложение   → только бизнес      │
│                                    │
│ Примечание: Близз создаётся        │
│ отдельно из строки Близзов.        │
└────────────────────────────────────┘
```

## Кнопки

| Кнопка | Кому видна | Функция | Куда ведёт |
|---|---|---|---|
| Активный аккаунт ▼ | всем | Открывает переключение аккаунтов | AccountSwitcherScreen |
| Пост | личный, business owner/admin/smm | Переход к будущему созданию поста | CreateActionIntroScreen(post) |
| Видео | личный, business owner/admin/smm | Переход к будущему созданию видео | CreateActionIntroScreen(video) |
| Предложение | только business owner/admin/smm | Переход к будущему созданию предложения | CreateActionIntroScreen(offer) |
| Открыть сообщения | только business messages | Открывает сообщения | MessagesScreen |

## Что запрещено на этом экране

```text
Близз
создание бизнеса
настройки
метрики
карта/геометка
витрина как отдельный раздел
медиа-редактор до согласования
публикация без согласованной страницы создания
```

## Реализация в версии 5

В версии 5 действия `Пост`, `Видео`, `Предложение` открывают короткий intro-экран будущего модуля. Это не редактор и не публикация, а рабочий переход, чтобы навигация не была мёртвой. Полные страницы создания будут согласованы отдельно.

## Связи

```text
MainTabs → + → CreateHubScreen
CreateHubScreen → AccountSwitcherScreen
CreateHubScreen → MessagesScreen для роли messages
CreateHubScreen → CreateActionIntroScreen(post/video/offer)
```
