# Страница / модуль 10 — Видео и вертикальная видеолента

## Назначение

Модуль добавляет две связанные части:

- `+ → Что создать? → Видео → CreateVideoScreen`;
- `Нижнее меню → Видео → VideoFeedScreen`.

Видео не смешивается с постами. Посты остаются фото+текст, видео — отдельный формат.

## Создание видео

Создание идёт только от `activeAccountId`.

Доступ:

- личный аккаунт — можно;
- бизнес `owner/admin/smm` — можно;
- бизнес `messages` — нельзя.

В версии 10 видео добавляется через URL:

- `videoUrl` — обязательная ссылка на видео;
- `coverUrl` — обязательная ссылка на обложку;
- `description` — описание;
- `location` — упрощённая геометка без карты;
- `visibility` — `public/followers/close_friends/selected`, где `selected` отключён до модуля приватности;
- `soundTitle` — пока `Оригинальный звук`.

Галерея, обрезка 9:16, первый кадр, media-picker, монтаж, фильтры, AR и библиотека музыки не добавлены.

## Видеолента

Вкладка `Видео` показывает опубликованные видео из backend:

- обложка;
- автор;
- описание;
- звук;
- геометка;
- ссылка на видео;
- лайк;
- сохранение.

Комментарии и Share для видео пока не подключены как полноценные модули, чтобы не обобщать прежние post-only модули без отдельного согласования.

## Профиль

Вкладка `Видео 0/1/...` в личном и бизнес-профиле показывает видео текущего `activeAccountId`.

Черновики профиля не смешиваются с видео: `Черновики` остаются невыложенными постами.

## Backend

Добавлены endpoints:

- `POST /api/videos`
- `GET /api/videos/feed`
- `GET /api/videos/my`
- `POST /api/videos/:videoId/like`
- `POST /api/videos/:videoId/save`

## Запрещено в модуле 10

- возвращать Близз в `+`;
- создавать видео ролью `business messages`;
- добавлять видео внутрь поста;
- добавлять media-picker без отдельного модуля;
- добавлять полноценные комментарии/share для видео без отдельного согласования;
- добавлять страницу звука и библиотеку музыки;
- смешивать видео-черновики с вкладкой `Черновики` профиля.
