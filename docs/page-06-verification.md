# Проверка версии 6

Дата: 2026-06-13.

## Backend

Пройдено:

```text
cd server
npm run check
node page06-smoke-test.js
```

Smoke-test проверяет:

- регистрацию;
- создание session;
- сохранение черновика поста;
- публикацию поста;
- `GET /api/posts/my`;
- `GET /api/posts/drafts`;
- обновление счётчиков `posts` и `drafts` в `GET /api/profile/me`;
- запрет публикации без фото;
- создание бизнес-аккаунта;
- публикацию поста от бизнес-аккаунта owner.

## Mobile

cd mobile

## Grep-проверки

Пройдено:

- нет `user.isBusiness`;
- нет нижней вкладки `Сообщения`;
- нет `Близз` как create-action;
- нет текста `Создайте первый пост, Близз или видео.`;
- нет блока `Быстрые разделы`;
- нет вкладки `Близзы 0` в профиле.
