# Проверка версии 3 — Профиль личного аккаунта

## Сверка с согласованной логикой

Проверено вручную по исходникам:

- нижнее меню: `Главная / Видео / + / Карта / Профиль`;
- `Сообщения` не являются вкладкой нижнего меню;
- кнопка личных сообщений есть в правом верхнем углу Главной и Профиля;
- в Профиле есть кнопка управления `☰`;
- верхний счётчик `Посты 0` отсутствует;
- верхняя статистика профиля содержит только `Подписчики` и `Подписки`;
- вкладки профиля: `Посты`, `Видео`, `Черновики`, `Сохранённое`;
- блок быстрых разделов отсутствует;
- подсказка `Создайте первый пост, Близз или видео.` не показывается в интерфейсе;
- пустое состояние публикаций: `Пока нет публикаций`.
