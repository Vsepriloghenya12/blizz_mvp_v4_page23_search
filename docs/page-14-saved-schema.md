# Page 14 — Сохранённое / Хочу сходить

## Назначение

`Хочу сходить` — приватный раздел сохранённых объектов активного аккаунта. Это не отдельная вкладка нижнего меню и не публичная подборка.

## Входы

- `Профиль → ☰ → Хочу сходить`.
- `Карта → фильтр Сохранённое`.
- Кнопки сохранения в постах, видео, предложениях и на карте.

## Что показывает

- Посты.
- Видео.
- Предложения.
- Бизнесы.

## Что не делаем в этом модуле

- Отдельную вкладку нижнего меню.
- Публичное сохранённое.
- Социальные подборки.
- Сложные маршруты внутри приложения.
- Автоопределение местоположения.
- Рейтинги, отзывы, бронирование и оплату.

## API

- `GET /api/saved?filter=all|post|video|offer|business`.
- `POST /api/saved`.
- `DELETE /api/saved/:targetType/:targetId`.

## Связь с картой

Фильтр `Карта → Сохранённое` использует тот же слой `savedItems`, что и экран `Хочу сходить`.
