# Page 17 — Игры в чатах / Угадай карту

## Решение пользователя

Из модуля игр полностью исключены:

- Квиз;
- Угадай место;
- Быстрый выбор / куда пойдём.

Их нет в UI, API catalog как готовых игр, backend create flow и карточках чата.

## Разрешённые игровые направления

- Угадай карту;
- Футбол;
- Напёрстки.

В версии 17 рабочей реализована только первая игра: `Угадай карту`.
`Футбол` и `Напёрстки` остаются отдельными будущими модулями, чтобы не делать несколько недоделанных игр одновременно.

## Вход

`Сообщения → Личный чат / Групповой чат → + → Игры`.

В бизнес-чате игры не показываются и backend запрещает запуск игры в business conversation.

## Backend

- `GET /api/games/catalog`
- `POST /api/games/sessions`
- `GET /api/games/sessions/:sessionId`
- `POST /api/games/sessions/:sessionId/answer`
- `POST /api/games/sessions/:sessionId/finish`

## Сообщения

При запуске игры создаётся сообщение `game_invite`.
После завершения создаётся сообщение `game_result`.

## Запреты

- нет денег;
- нет ставок;
- нет азартной механики;
- нет отдельной вкладки `Игры`;
- нет игр в бизнес-чате;
- нет Квиза;
- нет Угадай место;
- нет Быстрого выбора.
