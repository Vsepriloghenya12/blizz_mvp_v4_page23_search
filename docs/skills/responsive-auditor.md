# Responsive Auditor Skill — Blizz

Use before returning any archive after visual changes.

## Mobile Width

Check:

- 360 px width;
- 390 px width;
- 430 px width;
- notched/safe-area behavior;
- keyboard impact on forms;
- scroll content not clipped.

## Web/PWA Width

Check:

- desktop browser around 1280–1440 px;
- the mobile app does not stretch as if it were a desktop website;
- content max width is correct;
- hero images do not become accidental panoramic banners;
- bottom navigation remains phone-like.

## Required Notes

Every visual verification doc must include:

- tested widths;
- whether layout is centered/capped on web;
- known differences between native and web;
- screenshots/manual observation notes when possible.
