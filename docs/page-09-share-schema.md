# Страница / модуль 9 — Поделиться / Share

## Статус
Реализовано как продолжение версии 8 comments.

## Назначение
Share — общий модуль отправки контента. В этой версии подключён только к постам из Главной/ленты.

## Вход
Главная → карточка поста → кнопка `Поделиться`.

## UI
Share открывается светлым bottom sheet. Отдельный экран Share не создаётся.

## Действия
- `Поделиться` под постом открывает ShareSheet.
- `Закрыть` закрывает sheet.
- `Быстрые получатели` показывает собственные аккаунты пользователя, кроме активного аккаунта.
- `Отправить` создаёт `shared_content` через backend.
- `Скопировать ссылку` копирует/готовит `blizz://post/:postId`.
- `Поделиться вне приложения` использует web share, если доступен; иначе копирует ссылку.

## Backend
- `GET /api/share/recipients`
- `POST /api/share/post`

## Privacy
- `public` пост можно отправлять получателю.
- не-public пост пока не отправляется другому аккаунту до модуля приватности/подписок/близких.
- backend проверяет доступ отправителя и получателя.

## Shared content
Создаётся запись:

```json
{
  "type": "post",
  "contentId": "post_123",
  "senderAccountId": "account_sender",
  "targetType": "account",
  "targetAccountId": "account_target"
}
```

## Не реализовано в этом модуле
- полноценные чаты;
- создание групп;
- выбор нескольких получателей;
- share в Близз;
- публичный счётчик репостов;
- сложная приватность закрытого контента;
- отдельная страница Share.
