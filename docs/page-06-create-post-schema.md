# Страница 6 — Создание поста

## Назначение

`CreatePostScreen` открывается из `+ → Что создать? → Пост` и создаёт обычный пост от имени `activeAccountId`.

## Кто может создавать

| Аккаунт / роль | Доступ |
| --- | --- |
| personal | Да |
| business owner | Да |
| business admin | Да |
| business smm | Да |
| business messages | Нет |

## Схема перехода

```text
+ → Что создать? → Пост → Создание поста
                         ├─ Опубликовать → POST /api/posts → Профиль / Посты
                         └─ Сохранить черновик → POST /api/posts/drafts → Профиль / Черновики
```

## Поля

- Фото: 1–10 URL изображений для первой web-проверки.
- Текст: до 2200 символов.
- Геометка: название места и адрес, без карты на этом этапе.
- Видимость: Всем, Подписчикам, Близким, Выбранным. `Выбранным` disabled до модуля приватности.

## Кнопки

| Кнопка | Функция | Куда ведёт |
| --- | --- | --- |
| Назад | Возврат к `Что создать?` | CreateHubScreen |
| Активный аккаунт | Переключатель аккаунтов | AccountSwitcher |
| Добавить фото | Добавляет URL изображения | остаёмся на экране |
| Удалить | Удаляет URL изображения | остаёмся на экране |
| Опубликовать | Создаёт published post | ProfileScreen / Посты |
| Сохранить черновик | Создаёт draft post | ProfileScreen / Черновики |

## Backend

```text
POST /api/posts
GET /api/posts/my
POST /api/posts/drafts
GET /api/posts/drafts
```

Backend сам берёт `userId` и `activeAccountId` из session token. Клиент не отправляет `userId`.

## Запрещено на этой странице

- видео внутри поста;
- создание Близза;
- музыка;
- стикеры;
- AR/фильтры;
- отметки людей;
- автогеолокация без действия;
- создание бизнеса;
- публикация от `userId`.
