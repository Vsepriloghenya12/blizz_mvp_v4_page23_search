Три варианта красивого стиля

1. Blizz Soft Glyphs

рекомендую
message
like
place
save

Фирменный стиль: outline + мягкая заливка внутри ключевых иконок. Красиво, дружелюбно, не выглядит как системная библиотека.

  • Лучше всего подходит под теплый фон и deep blue.
  • Активные состояния становятся красивыми, а не просто цветными.
  • Можно реализовать в текущем `BlizzIcon` без новой зависимости.

2. City Badge Icons

более брендово
message
like
place
save

Иконка всегда в синем бейдже. Сильнее бренд, но есть риск утяжелить ленту, особенно если много действий под постами.

  • Хорошо для bottom nav и stories.
  • Плохо для плотной action-строки: слишком много синих пятен.
  • Можно использовать дозированно для primary actions.

3. Clean Premium Line

like

Самый спокойный вариант: красиво и premium, но слабее фирменность. Больше похож на Airbnb/современный SaaS.

  • Подойдет, если хотим очень дорогой и тихий интерфейс.
  • Меньше характера для социальной городской ленты.
  • Активные состояния нужно делать очень аккуратно.

Правила красоты для Blizz Soft Glyphs

ЗаливкаSoft-blue внутри важных иконок; danger fill только для active like.
Размер24px actions, 22px header, 28-30px stories/bottom nav center.
ЦветPrimary для active/brand, navy для обычных, muted для secondary.

Что берем в реализацию после approval
