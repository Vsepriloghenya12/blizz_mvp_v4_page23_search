Professional icon libraries

Выбираем не самодельный стиль, а профессиональную icon library

Для Близз нужен цельный продуктовый набор: выверенная сетка, одинаковый stroke, нормальная читаемость на iOS/Android/Web и большая библиотека под все будущие экраны. Самодельные иконки убираем из направления.

Lucide

рекомендую для проекта

Открыть каталог Lucide

  • Профессиональный, чистый, современный outline.
  • Есть официальный `lucide-react-native`.
  • Работает поверх `react-native-svg`, который уже есть в проекте.
  • Хорошо подходит для соцсети/городского приложения без визуального шума.

Tabler Icons

очень большой набор

Открыть каталог Tabler

  • Много иконок, аккуратная 24px сетка.
  • Хорош для dashboard/admin и утилитарных экранов.
  • Может выглядеть чуть более технически и менее mobile-social.
  • Есть React Native пакет, но я бы проверял совместимость отдельно.

Phosphor

самый выразительный

Открыть каталог Phosphor

  • Красивый, более дружелюбный и “живой” стиль.
  • Несколько весов: regular, bold, fill, duotone.
  • Визуально может лечь на Близз лучше Lucide.
  • Нужно аккуратно проверить React Native пакет и web/export.

Моя профессиональная рекомендация

Если цель Выбор Почему
Надежно внедрить сейчас Lucide Официальная поддержка React Native, уже совместимая основа `react-native-svg`, меньше риска сломать Expo web.
Максимально красиво и дружелюбно Phosphor Более мягкая пластика и несколько начертаний, но нужно сначала проверить пакет на текущем Expo SDK.
Больше утилитарности Tabler Очень большой набор, хорош для dashboards/settings, но для главной социальной ленты может быть суховат.

Стиль применения для Близз

Зона Иконки Правило
Header search, bell, message-circle 24px, stroke 2.1-2.25, круглая кнопка 44px, цвет `#071B3A` или `#0B3D99`.
Post actions heart, message-circle, send/share, bookmark 24px, без тяжелых подложек; active like красный, active save синий.
Bottom nav home, play, plus, map-pin, user 24px, active primary, inactive muted; центральный plus можно оставить крупнее.
Location/meta map-pin 14-16px inline, muted, чтобы не спорить с текстом и фото.
